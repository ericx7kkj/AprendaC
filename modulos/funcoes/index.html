Funções

Aprenda a criar funções para organizar e reutilizar seu código em C.

O que são funções?

Uma função é um bloco de código com nome próprio que pode ser chamado várias vezes. Elas tornam o código mais organizado, legível e reutilizável. O main() que você já conhece é uma função!

Estrutura de uma função

estrutura.c
// tipo_retorno  nome  (parâmetros)
int somar(int a, int b) {
    return a + b;      // retorna o resultado
}

int main() {
    int resultado = somar(3, 5);  // chama a função
    printf("%d\n", resultado);    // imprime 8
    return 0;
}

Funções void — sem retorno

void.c
void saudacao() {
    printf("Ola, seja bem-vindo!\n");
    // não precisa de return
}

int main() {
    saudacao();   // chama a função
    saudacao();   // chama de novo — reutilizável!
    return 0;
}

Protótipo de função

Em C, se a função é definida depois do main, você precisa declará-la antes (protótipo):

prototipo.c
#include <stdio.h>

int dobro(int n);   // protótipo — avisa que a função existe

int main() {
    printf("%d\n", dobro(5));
    return 0;
}

int dobro(int n) {    // definição completa abaixo
    return n * 2;
}
💡 Dica: para evitar protótipos, defina sempre suas funções antes do main().

Recursão e escopo de variáveis

recursao.c
// Recursão: função que chama a si mesma
int fatorial(int n) {
    if (n <= 1) return 1;   // condição de parada obrigatória!
    return n * fatorial(n - 1);
}
// fatorial(4) = 4*3*2*1 = 24

// Escopo: variável local só existe dentro da função
int x = 10;  // global — visível em todo o arquivo

void teste() {
    int x = 99;  // local — "esconde" a global dentro desta função
    printf("%d\n", x);  // imprime 99
}
⚠️ Toda recursão precisa de uma condição de parada, senão causa estouro de pilha (stack overflow).
