Variáveis e tipos de dados

Aprenda a declarar variáveis, entenda os tipos primitivos do C e como ler e imprimir dados.

O que são variáveis?

Uma variável é um espaço nomeado na memória RAM onde guardamos um valor durante a execução do programa. Em C, toda variável precisa ser declarada com um tipo antes de ser usada.

Declarando e inicializando variáveis

declaracao.c
int idade = 20;
float altura = 1.75;
double pi = 3.14159265358979;

// CUIDADO: declarar sem inicializar = lixo de memória

x = 10;  // agora está seguro

scanf() — Lendo dados do usuário

leitura.c
int idade;
printf("Digite sua idade: ");
scanf("%d", &idade);   // & é obrigatório!
printf("Voce tem %d anos.\n", idade);

Operadores aritméticos

operadores.c
int a = 10, b = 3;
printf("%d\n", a + b);   // 13 — soma
printf("%d\n", a - b);   // 7  — subtração
printf("%d\n", a * b);   // 30 — multiplicação
printf("%d\n", a / b);   // 3  — divisão INTEIRA
printf("%d\n", a % b);   // 1  — resto da divisão

Constantes com #define e const

constantes.c
#include <stdio.h>
#define PI 3.14159      // substituição textual pelo pré-processador
#define MAX 100

int main() {
    const int LIMITE = 50;  // const: variável que não pode mudar
    float area = PI * 5.0 * 5.0;
    printf("Area: %.2f\n", area);   // Area: 78.54
    printf("Max: %d\n", MAX);
    printf("Limite: %d\n", LIMITE);
    return 0;
}
💡 Use #define para valores que nunca mudam e const quando quiser que o compilador verifique o tipo da constante.
