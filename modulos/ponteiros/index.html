Ponteiros

Entenda um dos conceitos mais poderosos — e temidos — da linguagem C: ponteiros e endereços de memória.

O que é um ponteiro?

Toda variável em C ocupa um espaço na memória RAM e esse espaço tem um endereço — como o número de uma casa numa rua. Um ponteiro é uma variável que armazena esse endereço, ou seja, ele "aponta" para outro lugar na memória.

ponteiro_basico.c
int x = 10;       // variável comum
int *p = &x;     // p aponta para x (&x = endereço de x)

printf("%d\n",  x);   // 10  — valor de x
printf("%p\n",  p);   // 0x... — endereço de x
printf("%d\n", *p);   // 10  — valor apontado por p
OperadorNomeO que fazExemplo
&Endereço deRetorna o endereço da variável&x
* (declaração)PonteiroDeclara uma variável ponteiroint *p
* (uso)DesreferênciaAcessa o valor no endereço*p
💡 O * tem dois papéis diferentes: na declaração (int *p) ele diz que p é um ponteiro. No uso (*p) ele acessa o valor apontado. Não confunda os dois!

Modificando variáveis via ponteiro

A grande utilidade dos ponteiros é poder modificar o valor original de uma variável de dentro de uma função — algo impossível com passagem por valor normal.

modificar.c
#include <stdio.h>

void dobrar(int *n) {
    *n = *n * 2;   // modifica o valor ORIGINAL
}

int main() {
    int x = 5;
    dobrar(&x);          // passa o endereço de x
    printf("%d\n", x);  // imprime 10
    return 0;
}

Ponteiros e arrays

Em C, o nome de um array é na verdade um ponteiro para o primeiro elemento. Por isso ponteiros e arrays andam juntos:

array_ponteiro.c
int nums[] = {10, 20, 30};
int *p = nums;   // p aponta para o primeiro elemento

printf("%d\n", *p);       // 10 — primeiro elemento
printf("%d\n", *(p+1));   // 20 — segundo elemento
printf("%d\n", *(p+2));   // 30 — terceiro elemento

// Equivalente a usar índices:
printf("%d\n", nums[0]);   // 10
printf("%d\n", nums[1]);   // 20

Ponteiro nulo e boas práticas

boas_praticas.c
#include <stdio.h>

int main() {
    int *p = NULL;   // ponteiro nulo — não aponta para nada

    if (p == NULL) {
        printf("Ponteiro nao inicializado!\n");
    }

    int x = 42;
    p = &x;          // agora p aponta para x
    printf("%d\n", *p);   // 42 — seguro usar agora
    return 0;
}
⚠️ Nunca use um ponteiro sem inicializá-lo. Um ponteiro não inicializado aponta para um endereço aleatório — acessá-lo causa comportamento indefinido e pode travar o programa.

Aritmética de ponteiros

Ponteiros suportam operações aritméticas. Somar 1 a um ponteiro avança para o próximo elemento do tipo na memória, não para o próximo byte:

aritmetica.c
int v[] = {1, 2, 3, 4, 5};
int *p = v;

// percorrendo o array com ponteiro
for (int i = 0; i < 5; i++) {
    printf("%d ", *(p + i));
}
// saída: 1 2 3 4 5

// p++ avança p para o próximo int
p++;
printf("\n%d\n", *p);   // 2
