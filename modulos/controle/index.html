Estruturas de controle

Aprenda a controlar o fluxo do seu programa com condicionais e laços de repetição.

Condicionais — if / else

Permitem executar blocos de código apenas se uma condição for verdadeira.

condicional.c
int nota = 7;
if (nota >= 7) {
    printf("Aprovado!\n");
} else if (nota >= 5) {
    printf("Recuperacao\n");
} else {
    printf("Reprovado\n");
}
OperadorSignificadoExemplo
==Igual aa == b
!=Diferente dea != b
>Maior quea > b
<Menor quea < b
>=Maior ou iguala >= b
<=Menor ou iguala <= b
⚠️ Cuidado: = é atribuição, == é comparação! Usar = dentro de um if é um erro clássico.

switch / case

Útil quando se quer testar um valor contra várias opções fixas.

switch.c
int dia = 2;
switch (dia) {
    case 1: printf("Segunda\n"); break;
    case 2: printf("Terca\n");   break;
    case 3: printf("Quarta\n");  break;
    default: printf("Outro dia\n");
}

Laço for

Usado quando se sabe o número de repetições. Sintaxe: for (início; condição; incremento).

for.c
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
// Imprime: 1 2 3 4 5 (um por linha)

Laço while

Repete enquanto a condição for verdadeira. Ideal quando não se sabe o número de repetições.

while.c
int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}

do / while

Similar ao while, mas garante que o bloco executa pelo menos uma vez.

dowhile.c
int i = 1;
do {
    printf("%d\n", i);
    i++;
} while (i <= 5);

break, continue e laços aninhados

controle_extra.c
// break — encerra o laço imediatamente
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    printf("%d ", i);  // 0 1 2 3 4
}

// continue — pula para próxima iteração
for (int i = 0; i < 6; i++) {
    if (i % 2 == 0) continue;
    printf("%d ", i);  // 1 3 5
}

// laços aninhados
for (int i = 1; i <= 3; i++)
    for (int j = 1; j <= 3; j++)
        printf("%d*%d=%d\n", i, j, i*j);
⚠️ break dentro de laço aninhado encerra apenas o laço mais interno.
